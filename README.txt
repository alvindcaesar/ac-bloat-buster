A super simple plugin to remove unnecessary bloat from your WordPress site.

== Description ==

Features:
* Remove Emoji Scripts - Remove the emoji.js if you don\'t use emoji on your site.

* Remove Full Site Editing Global Styles - This will prevent the FSE global CSS from loading because this script is enabled by default even if you\'re not using FSE.
